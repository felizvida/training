## Exercise 4: Restart Handoff

Expected components:

- Goal
- Current state
- Key source files
- Commands or checks run
- Outputs produced
- Verification status
- Privacy and source boundary
- Known limitations
- Next steps

Instructor note: The handoff should preserve retrieval handles such as paths and commands. It should not include fake secrets, invented hostnames, or row-level source details.
