# Capstone: Complete A Scientific-Support Task

## Scenario

Choose one of the synthetic datasets in [../data](../data). Your job is to complete a small staff-facing task from request to handoff.

## Requirements

Your Codex-assisted work must include:

- A clear request with source, audience, privacy boundary, and expected output.
- A concise staff-facing result.
- Evidence notes naming the source fields or records used.
- At least one verification step.
- A short restart handoff.

## Suggested Task Options

### Option A: Software Inventory

Use [../data/scientific_software_inventory.csv](../data/scientific_software_inventory.csv) to produce a ranked aggregate summary of scientific software families.

### Option B: Renewal Snapshot

Use [../data/renewal_projection_snapshot.csv](../data/renewal_projection_snapshot.csv) to answer what remains to order for true software renewals.

### Option C: Role Resolution

Use [../data/identity_lookup_samples.csv](../data/identity_lookup_samples.csv) to answer who appears responsible for each support function while preserving ambiguous or unresolved records.

## Deliverables

1. Staff-facing answer.
2. Verification note.
3. Restart handoff.

## Review Questions

- Did the final output answer the practical question?
- Did it avoid exposing unnecessary identifiers?
- Did it name the evidence used?
- Did it keep uncertainty visible?
- Could another person continue from the handoff?
