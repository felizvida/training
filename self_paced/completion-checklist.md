# Completion Checklist

Use this before marking the 8-unit tutorial complete.

## Required Deliverables

- [ ] Progress tracker completed.
- [ ] Unit 1 task inventory completed.
- [ ] Unit 2 aggregate scientific software summary completed.
- [ ] Unit 3 privacy and verification checklist completed.
- [ ] Unit 4 reconciliation or renewal answer completed.
- [ ] Unit 5 workflow status report completed.
- [ ] Unit 6 identity or role-resolution answer completed.
- [ ] Unit 7 handoff or source memo completed.
- [ ] Unit 8 capstone task package completed.

## Quality Checks

- [ ] Each deliverable names its source evidence.
- [ ] Outputs avoid unnecessary row-level identifiers.
- [ ] Uncertainty is preserved where evidence is weak.
- [ ] Verification steps are recorded.
- [ ] The capstone includes what Codex should not assume.
- [ ] The final handoff is reusable by another person.

## Shareability Checks

- [ ] No real secrets, passwords, tokens, private notes, or ticket exports.
- [ ] No real names, emails, hostnames, or device IDs unless explicitly authorized for a restricted audience.
- [ ] Synthetic data is clearly synthetic.
- [ ] Any real-work follow-up is labeled as needing local authorization and human review.
