# Unit 4: Reconciliation And Renewal Questions

Time: 1 hour

## Goal

Answer comparison and renewal questions without mixing categories or overstating conclusions.

## Anchored Experience

This unit combines two recurring patterns:

- Ask "what new evidence do we learn?" when comparing sources.
- Separate true renewal/order items from one-time adjustments, discontinued items, and non-software costs.

## 60-Minute Plan

| Time | Activity |
| --- | --- |
| 0-10 min | Inspect renewal categories and statuses |
| 10-20 min | Identify true renewal/order rows |
| 20-35 min | Draft or run the renewal answer prompt |
| 35-45 min | Compare to answer key |
| 45-60 min | Write a brief caveat and verification note |

## Starter Prompt

```text
Use data/renewal_projection_snapshot.csv to answer what remains this fiscal year for true software renewal/order work.
Separate renewal/order rows from one-time adjustments, discontinued items, ordered items, and non-software costs.
Give a concise table, subtotal, excluded categories, source artifact, and assumptions.
```

## Deliverable

A concise renewal answer with:

- True renewal/order remaining.
- Excluded categories.
- Source artifact.
- Verification calculation.

## Self-Check

- Did you avoid using a raw total that mixes categories?
- Did you state what was excluded?
- Did you verify the subtotal?
- Did you preserve caveats?
