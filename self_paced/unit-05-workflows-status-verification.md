## Starter Prompt

```text
Inspect this training repository and produce a status report for the self-paced tutorial package.
Use actual files as evidence.
Report current state, files checked, validation that should be run, gaps, and next action.
Do not infer that something is complete unless the repository evidence supports it.
```

## Deliverable

A status report with:

- Current state.
- Evidence checked.
- Validation or checks.
- Gaps or risks.
- Next action.

## Self-Check

- Did the report name actual files?
- Did it separate evidence from inference?
- Did it include a verification step?
- Did it include a concrete next action?
