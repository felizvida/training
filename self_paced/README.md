# Self-Paced Tutorial: 8 Units, 1 Hour Each

This self-paced tutorial is a complete 8-hour delivery modality for scientific staff learning to use Codex on local, evidence-based work.

It is anchored on the package owner's project patterns:

- Scientific software inventory and family-collapsed summaries.
- Multi-source reconciliation and "what new evidence did we learn?" comparisons.
- Local data refreshes with documented verification.
- Renewal planning that separates true action items from adjustments.
- Workflow status checks grounded in actual state.
- Local-first identity and role lookup with unresolved cases kept explicit.
- App usability, testing, documentation, handoffs, source memos, and release notes.
- This training repository's own maintenance as a Codex-assisted workflow.

Brief examples for other scientific staff and researchers appear only as optional context.

## Who This Is For

Learners can complete the tutorial alone, with or without an instructor present. No programming experience is required, though Units 5, 7, and 8 introduce repo-aware habits.

## What Learners Need

- This repository.
- A Markdown viewer or text editor.
- The synthetic CSV files in [../data](../data).
- Optional: access to Codex for live practice.

If a learner does not have live Codex access, they can still complete the tutorial by writing the prompts, expected outputs, verification plans, and handoffs manually.

## How To Use This Track

1. Read [learner-guide.md](learner-guide.md).
2. Complete units in order.
3. Save each unit deliverable in a personal notes file or local folder.
4. Use [portfolio-template.md](portfolio-template.md) to collect final artifacts.
5. Use [completion-checklist.md](completion-checklist.md) before marking the tutorial complete.

## Completion Standard

A learner completes the tutorial when they can show:

- One source-grounded summary.
- One privacy-boundary checklist.
- One comparison or renewal answer.
- One status or verification report.
- One uncertainty-preserving identity/role answer.
- One handoff or source memo.
- One final Codex-ready task plan grounded in their own work.
